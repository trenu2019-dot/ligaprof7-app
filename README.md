# LigaPro F7 V32 — PWA Instalable

Esta versión convierte la demo en una PWA instalable.

## Qué incluye

- Manifest PWA completo.
- Íconos 192/512 y maskable.
- Service worker.
- Pantalla offline.
- Botón "Instalar app".
- Guía `/instalar`.
- Compatibilidad con Android y iPhone.
- Carga Excel/CSV de V30.4.2.
- Guía celular de V31.
- Reportes, QR y backup.

## Abrir localmente

Ejecuta:

`INICIAR_APP_V32_PWA.bat`

Abre:

`http://127.0.0.1:8056`

Guía instalación:

`http://127.0.0.1:8056/instalar`

Guía celular:

`http://127.0.0.1:8056/celular`

## Usuarios

- admin / 2026
- arbitro / 7777
- tigres / 1111
- consulta / 0000

## Nota importante

En celular, una PWA se instala correctamente cuando está publicada en HTTPS.  
En red local puedes probarla y abrirla como app web, pero para instalación formal se recomienda publicarla en internet.

## Siguiente paso recomendado

Subir esta V32 a Render, Railway, VPS o dominio propio con HTTPS.
