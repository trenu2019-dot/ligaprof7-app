# LigaPro F7 V1.6 — Deportstar UX

Esta versión ya no solo cambia colores. Reestructura la pantalla principal para parecerse más a la referencia tipo Deportstar.

## Cambios principales

- Barra superior clara con navegación: Explorar, Fixture, Comunidad, Guías.
- Portada tipo plataforma con explicación de cómo empezar.
- Bloques de pasos: Crear torneo, Inscribir equipos, Generar partidos, Registrar resultados.
- Sección “Cómo funciona” con tarjetas informativas.
- Panel operativo separado, menos saturado.
- Login en ventana modal, no mezclado en toda la pantalla.
- Menú de módulos reducido y ordenado.
- Mantiene funcionalidades estables.
