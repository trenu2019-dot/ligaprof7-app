# LigaPro F7 V1.3 — UI Premium Dashboard

Versión preparada para GitHub y Render con el diseño solicitado:
- Logo F7 premium.
- Encabezado tipo app profesional.
- Tarjeta de sesión premium.
- Menú principal en dashboard de tarjetas.
- Íconos, bordes, sombras y acentos verde/dorado.
- Barra inferior tipo app.
- Módulos: Inicio, Equipos, Jugadores, Calendario, Resultados, Tabla, Pagos, Reportes, Carga masiva y QR/Credenciales.

## Subir a GitHub y Render

1. Descomprime este ZIP.
2. Sube TODO el contenido a la raíz del repositorio de GitHub.
3. Commit changes.
4. En Render: Manual Deploy > Clear build cache & deploy.

## Verificar

Abre:

https://ligaprof7-app.onrender.com/api/version

Debe responder:

V1_3_UI_PREMIUM_DASHBOARD

Después abre la APK o la URL pública.
Si el celular guarda versión anterior: Ajustes > Apps > LigaPro F7 > Almacenamiento > Borrar caché y datos.
