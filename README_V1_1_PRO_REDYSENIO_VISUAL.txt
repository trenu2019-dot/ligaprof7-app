# LigaPro F7 V1.1 Pro — Rediseño Visual Real

Esta versión sí cambia visualmente la experiencia de la app, especialmente en celular.

## Cambios visibles

- Pantalla móvil rediseñada tipo app.
- Menú convertido en tarjetas con iconos.
- Encabezado más profesional.
- Badges de APK, reportes, roles y torneo activo.
- Dashboard móvil con estado online y versión.
- Paneles con mayor profundidad visual.
- Módulos renombrados con lenguaje más comercial.
- Versión pública: V1_1_PRO_REDYSENIO_VISUAL.

## Subir a GitHub y Render

1. Sube todo el contenido de esta carpeta a GitHub.
2. En Render: Manual Deploy > Clear build cache & deploy.
3. Verifica:

https://ligaprof7-app.onrender.com/api/version

Debe responder:

V1_1_PRO_REDYSENIO_VISUAL

## En celular

Después de Render, cierra la APK y ábrela de nuevo.
Si ves versión anterior, borra caché/datos de LigaPro F7.
