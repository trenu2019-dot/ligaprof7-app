# LigaPro F7 V1.2 — App con Pantallas Reales

Esta versión cambia la navegación móvil para que se sienta como una app profesional.

## Cambio principal

Antes:
- Menú y contenido estaban en una sola página.
- Tocabas un botón y tenías que desplazarte hacia abajo manualmente.

Ahora:
- En celular se muestra un menú principal tipo app.
- Al tocar un módulo, se abre una pantalla completa independiente.
- Cada pantalla tiene barra superior con botón "← Menú".
- La pantalla tiene navegación inferior rápida.
- Ya no hay que bajar manualmente para ver el módulo.

## Verificación

Después de subir a GitHub y Render, abre:

https://ligaprof7-app.onrender.com/api/version

Debe responder:

V1_2_APP_PANTALLAS_REALES

## Uso

1. Sube esta carpeta completa a GitHub.
2. En Render ejecuta: Manual Deploy > Clear build cache & deploy.
3. En el celular abre la APK.
4. Si sigue la versión anterior, borra caché/datos de LigaPro F7.
